
# 🐱 网页逗猫小球 (Cat Toy Web App)

一个基于 HTML5 Canvas 的轻量级逗猫网页应用。模拟一只带有尾巴的小球在屏幕上跑动、躲藏和探头，旨在吸引猫咪的注意力。采用童趣的手绘风格（Doodle Style）UI 设计。

## ✨ 主要功能

*   **🐁 拟真生物行为**
    *   小球会变速跑动、停顿发呆。
    *   会跑出屏幕边界“躲藏”，过一会从随机边缘偷偷“探头”观察。
    *   防死循环算法，确保小球移动流畅。
*   **⚡ 互动反馈**
    *   支持触摸（手机/平板）和鼠标点击。
    *   被抓到时，小球会剧烈**震动挣扎**并发出**吱吱声**（使用 Web Audio API，无需额外素材）。
*   **🎨 个性化配置**
    *   **手绘风格 UI**：齿轮、按钮和面板均采用涂鸦风格。
    *   **参数调节**：可调整小球的移动速度、体积大小以及被抓后的挣扎时长。
    *   **主题切换**：内置 6 种配色方案（包括经典米白、护眼暗夜、高对比度红球等）。
*   **🔒 安全锁定模式 (猫锁)**
    *   右上角一键锁定并全屏，隐藏所有干扰 UI。
    *   防止猫爪误触导致退出或切换页面。
    *   **解锁方式**：长按右上角解锁按钮 2 秒即可恢复。

## 🚀 如何使用

1.  将源码保存为 `.html` 文件（例如 `cat_toy.html`） 或直接访问我的网站 [这里](https://ball.catflix.cn)。
2.  直接使用现代浏览器（Chrome, Safari, Edge 等）打开该文件。
3.  建议在 **iPad** 或 **大屏手机** 上全屏运行，效果最佳。

## 🛠️ 技术栈

*   原生 HTML / CSS / JavaScript
*   HTML5 Canvas (绘图)
*   Web Audio API (音效)
*   无任何第三方库依赖，单文件即可运行。
